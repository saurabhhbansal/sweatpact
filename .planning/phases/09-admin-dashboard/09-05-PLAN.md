<objective>
Build the three PostHog-backed presentation panels for DASH-04 (onboarding funnel), DASH-05 (feature adoption), DASH-06 (engagement & retention). Each is a props-only component that renders metrics when data is present and the locked empty state when its rows prop is `null`/empty (PostHog data only began accumulating 2026-06-28, and the key may be unset).

Purpose: Per-panel empty-state independence is what lets the admin page render gracefully whether or not PostHog is reachable.
Output: 3 components in `src/components/admin/`. Visual correctness verified at phase UAT; typecheck verifies compilation.
</objective>
